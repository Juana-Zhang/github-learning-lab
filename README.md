# GitHub Learning Lab

This repository is a lightweight learning journal for regular GitHub activity.
It is separate from portfolio projects, so experiments, notes, and small weekly
updates can stay organized without changing finished work.

## Focus Areas

- AI agents and assistant workflows
- Python data analysis
- Prompt design and evaluation
- RAG fundamentals
- Testing and reproducibility
- GitHub Actions and automation
- Docker basics

## Weekly Rhythm

Target: 3 to 4 meaningful updates per week.

Good update examples:

- Add a short technical note
- Improve a code snippet
- Summarize one concept with references
- Add a tiny experiment
- Add or update a test
- Write a weekly learning reflection
