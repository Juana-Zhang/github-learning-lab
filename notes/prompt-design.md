# Prompt Design

Prompt design is the practice of giving an AI system enough context,
constraints, and examples to produce useful output.

For data analysis tasks, good prompts usually include:

- The business question
- The available data columns
- The expected output format
- Constraints about assumptions
- A request to show uncertainty or caveats

## Template

```text
Goal:
Data:
Constraints:
Output format:
Quality checks:
```

## Habit

When a prompt works well, save it with a short note explaining why it worked.
When it fails, save the failure mode too.
