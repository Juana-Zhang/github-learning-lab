# AI Agents

An AI agent is a system that uses a model to plan, use tools, remember context,
and take steps toward a goal.

For a learning portfolio, the useful angle is not just "the agent can do
things." The stronger angle is whether the workflow is understandable,
testable, and safe to run.

## Concepts To Track

- Goal decomposition
- Tool use
- Memory and context management
- Human approval points
- Evaluation and logging
- Failure recovery

## Questions

- What should the agent be allowed to do automatically?
- What should require human review?
- How can the output be checked?
- What logs would help debug a bad answer?
